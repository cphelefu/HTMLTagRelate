Example 1 Change by action [data-relate-action]

Code of Child Input 

								<input type="text" class="form-control" id="child1" 
data-relate-parent="#parent1" 

								data-relate-values="html,tag,relate"
 data-relate-action="disable" />

Description: relate-action defines the actions you want to perform when a relationship is validated i.e. when the values entered in the parent element match the values in the data-relate-values tag. In the above example we disable the child element.
Possible values of data-relate-action: hide, show, disable, checked(for checkboxes only), hideoption(for select tag only).

Example 3: Changing attributes in a relationship [relate-attrib, relate-attribvalue-true, relate-attribvalue-false] tag

Code of Child Input 

								<input type="text" class="form-control" id="child3" 
data-relate-parent="#parent3" 

								data-relate-values="html,tag,relate" 
data-relate-attrib="style" 
data-relate-attribvalue-true="border:4px solid Red"
data-relate-attribvalue-false="border:2px inset" />

Description: Sometimes you want to change the attribute/property on an element based on the input of another element. HTMLTagRelate.js uses the following tags to do that: relate-attrib, relate-attribvalue-true, relate-attribvalue-false. As you may have guessed relate-attrib identifies the attribute you want to change, relate-attribvalue-true defines what you want to set the attribute to when the relationship evaluates to true, relate-attribvalue-false defines what you want to set the chosen attribute to if the relationship evaluates to false.
Possible values of relate-attrib: All possible html attributes. e.g. class, style, width, height etc.

Example 5 Working with Select boxesJust working with select boxes

Code of Child Input 

								<select id="child5" 
								
action="hideoption"
								
data-relate-parent="#parent5" 

								<option data-parent-values="html,tag" value="option1">html,tag</option> 

								    <option data-parent-values="html,relate" value="option2">html,relate</option> 

								    <option data-parent-values="tag,relate" value="option3">tag,relate</option> 

								</select>

Description: This example just shows how to use HTMLTagRelate with the select element. In the above example when you change selected values the child select/dropdown will also filter to show options that have data-parent-values that correspond to value in the parent element.

Example 6 Introducing functions Changing attributes in a relationship [relate-attrib, relate-attribvalue-true, relate-attribvalue-false] tag

Code of Child Input 

								<div id="child6" 
data-relate-parent="#parent6" 

								data-relate-values="html,tag,relate"
data-relate-function-true="growIt()"
data-relate-function-false="reduceIt()"> <div/>

Description: In case you want to call a function when certain value are entered; this is the example for you! HTMLTagRelate.js the tag : relate-function to handle this - just specifify the name of the function you would like to call when the relationship is validated.
